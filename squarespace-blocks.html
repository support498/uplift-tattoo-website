How to Use These Components in Squarespace

Each component is a self-contained HTML+CSS block. Paste it into a Code Block on any Squarespace page - no theme editing required. Each one works independently.

  1. In the Squarespace editor click + to add a block
  2. Choose Code from the block menu
  3. Paste the component code and click Apply
  4. Replace YOUR_IMAGE_URL with your real photo URLs
  5. Drag code blocks up/down in the editor to reorder sections

Images: Upload photos to Squarespace first, right-click the image and copy the URL. Paste in place of YOUR_IMAGE_URL. All images automatically display in black and white to match the UPLIFT brand.

Publishing on Adobe Portfolio - Step by Step

Your 2,051 images are already in Creative Cloud. They sync to Portfolio instantly - no re-uploading.

1

Go to portfolio.adobe.com

Sign in with your Adobe ID - the same login you use for Creative Cloud.

2

Pick a clean minimal theme

Choose "Wright", "Bor", or "Slide" - white or black background. Avoid busy templates. Match UPLIFT's editorial aesthetic.

3

Create a page per artist

Add Page > Image Gallery for each artist (Junh, Anastasia, Julia, etc.). This becomes their individual portfolio page.

4

Sync from Creative Cloud

Inside each gallery, click Add Images > Creative Cloud. Your "Uplift Tattoo Portfolio" folder and all 2,051 images appear. Select and sync - no re-upload needed.

5

Set your domain

In Portfolio Settings > Custom Domain, use portfolio.uplifttattoo.com or a standalone domain. Update your DNS to point to Portfolio.

6

Link to your main site

Add a header link or button pointing back to uplifttattoo.com so clients can book from Portfolio.
