UPLIFT - Squarespace Deployment Guide

Add every page to your live Squarespace site

How to Add Each Page to Squarespace

For every page below, follow these 4 steps in your Squarespace admin panel:

  1. Pages → + Add Page → Blank Page — set the URL slug shown
  2. Add a Code Block for each section listed — paste the code from the matching file in squarespace-pages/
  3. Add the JSON-LD Schema — go to Pages → Settings → Advanced → Page Header Code Injection and paste the schema
  4. Replace image placeholders — upload photos to Squarespace, copy the URL, replace YOUR_IMAGE_URL

Open the matching .html file in this folder in your browser. Each section is separated and labeled. Copy one section at a time into a Squarespace Code Block.
