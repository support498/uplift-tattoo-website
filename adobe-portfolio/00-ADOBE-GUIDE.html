UPLIFT — Adobe Portfolio

Setup guide for portfolio.adobe.com — artist showcase site alongside uplifttattoo.com

What this site is for

Adobe Portfolio is a clean, visual-first site that comes free with Creative Cloud. UPLIFT's Adobe Portfolio is an artist showcase — full-bleed gallery pages for each artist, with a booking CTA that links back to uplifttattoo.com. It is not a replacement for the Squarespace site.

Think of it as: Squarespace = service/SEO/booking site. Adobe Portfolio = visual portfolio and artist discovery.

One-time setup (15 minutes)

1
Go to portfolio.adobe.com
Sign in with your Adobe Creative Cloud account. Click "Create Website."
2
Choose a theme
See recommended themes below. Pick one before adding any content — changing themes later resets your layout.
3
Set site title and URL
Site name: UPLIFT Tattoo & Piercing — Artist Portfolios. Default URL: upliftnyc.myportfolio.com (or connect custom domain: portfolio.uplifttattoo.com)
4
Create one page per artist
Use the copy from each artist file in this folder. Each page = gallery + short bio + "Book with [Name]" CTA.
5
Create a Cover / Home page
Use home.html in this folder for copy. Add a hero image (studio shot or collage of work). Link to uplifttattoo.com.
6
Publish and add link to Squarespace
Once live, add the Adobe Portfolio URL to the Squarespace footer nav under "Artists."

Recommended themes

Alba
Clean, minimal, full-bleed images. Best for dark photo sets. Top pick for UPLIFT.
Bram
Dark background, single-column layout. Very editorial. Works well for fineline close-ups.
Slate
Grid-based, neutral. Good if you want all artists on one visual grid page.
Helios
Modern sans-serif headers. Good for mixed tattoo + piercing content.
Tip

Preview Alba and Bram with a dark studio photo before deciding. Both support full-bleed hero images which suit UPLIFT's aesthetic best.

Image guidelines for Adobe Portfolio

→
Per artist: 8–16 photos minimum
Mix of healed work, fresh work, and close-up detail shots. Square or portrait orientation preferred.
→
File size: 2MB or under per image
Adobe Portfolio resizes on upload but smaller files load faster.
→
No watermarks, no text overlays
The portfolio theme handles labels. Clean photos only.
→
Cover/hero image per page
One strong, high-contrast image for each artist's page hero. Often a close-up of their signature style.
